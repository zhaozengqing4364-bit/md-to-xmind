# 深度学习入门

## 第一部分 基础篇

### 神经网络原理

神经网络由多层神经元组成。

> 神经网络是深度学习的基础。

#### 前向传播

输入数据从输入层经过隐藏层传递到输出层。

```
z = W * x + b
a = sigmoid(z)
```

#### 反向传播

通过链式法则计算梯度。

| 概念 | 说明 |
|------|------|
| 损失函数 | 衡量预测误差 |
| 梯度下降 | 沿梯度反方向更新参数 |

### 常见网络层

- 全连接层
- 卷积层
  - 1D卷积（序列数据）
  - 2D卷积（图像数据）
- 池化层
- Dropout层
